Introduction

On this page you will learn about plenty of the most useful java tools, that you will need to know for the rest of the course. Here is what you will learn:

  • If statements
  • If-else statements
  • While loops
  • Do-while loops
  • For loops
If statements

The syntax for an if statement looks like this:


if(boolean statement){
  more code;
}

This allows for the code inside the loop to only be run if the boolean in the if statement is true.

If-else statements

Building off of the regular if statement is the if-else statement, which has a syntax like this:


if(boolean statement){
  more code;
}else{
  more code;
}

With the if-else loop, if the boolean in the if statement is false, then the code after the else statement will be run, otherwise it will only run the code after the if statement.

While loops

A while loop is a loop that will continuously loop over the same code as long as a certain boolean value is true. This is the syntax for a while loop:


while(boolean statement){
  more code;
}

The code inside the loop will only run as long as the boolean statement before hand is checked and has a value of true.

Do-while loops

Do-while loops are just like normal while loops, except they check whether the boolean statement is true after running the code in the loop. Here is the syntax for a do-while loop:


do{
  more code;
}while(boolean statement);

This makes it so that the code inside the loop is run at least once, before checking if it should be run again.

For loops

A for loop makes loops a lot easier by automatically incrementing a newly created integer and using it to determine when to stop the loop. This is a common syntax for a for loop:


for(int i = 0; i < n; i++){
  more code;
}

This for loop makes it so that the newly created integer i equals 0, and the loop will run n times before stopping the loop.
